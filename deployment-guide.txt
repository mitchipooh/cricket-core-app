
CRICKET-CORE 2026 WORDPRESS DEPLOYMENT GUIDE
============================================

Follow these steps to put the app on your site:

1. BUILD THE APP:
   In your local terminal, run:
   npm run build

2. PREPARE THE PLUGIN:
   - Go to your WordPress server: wp-content/plugins/
   - Create a folder named: cricket-core-bridge
   - Copy 'wp-cricket-core.php' into this folder.
   - Create a subfolder inside it named 'dist'.
   - Copy all files from your local 'dist' folder into this new server 'dist' folder.

3. ACTIVATE:
   - Go to WordPress Admin -> Plugins.
   - Find "Cricket-Core 2026 Management Bridge" and click Activate.

4. FLUSH PERMALINKS (Required for Standalone Mode):
   - Go to WordPress Admin -> Settings -> Permalinks.
   - Scroll down and click "Save Changes" (no need to change any settings, just saving flushes the rules).

5. USAGE:
   
   Option A: Embed in a Page (Standard)
   - Create a new Page in WordPress.
   - Add a "Shortcode" block.
   - Enter: [cricket_core_app]
   - Publish.

   Option B: Standalone App (App Experience)
   - Navigate to: https://your-site.com/cricket-app/
   - This URL loads the app in full-screen mode without your WordPress theme header/footer.
   - Perfect for "Add to Home Screen" usage on mobile.

The app will now load, automatically styled and connected to your database!
